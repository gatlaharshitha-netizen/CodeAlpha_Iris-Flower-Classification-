# Iris Flower Classification using Machine Learning

## Description
This project classifies Iris flowers into one of three species—Setosa, Versicolor, and Virginica—using Machine Learning. The model is trained using the Iris dataset and a Decision Tree Classifier to predict the species based on four flower measurements.

## Features
- Load and analyze the Iris dataset
- Data preprocessing
- Data visualization using graphs
- Decision Tree model training
- Species prediction
- Accuracy evaluation
- Confusion matrix generation
- Classification report

## Dataset
The dataset contains **150 flower samples** with the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

**Target Classes:**
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm
**Decision Tree Classifier**

## How to Run
1. Install the required Python libraries.
2. Place the `Iris.csv` file in the project folder.
3. Run the Python program:
   ```bash
   python iris_classification.py
   ```
4. View the graphs, model accuracy, confusion matrix, classification report, and predicted species.

## Output
- Displays the first five rows of the dataset.
- Shows dataset information and missing values.
- Visualizes species distribution using a count plot.
- Generates a pair plot and correlation heatmap.
- Trains the Decision Tree model.
- Predicts the species of a new Iris flower.
- Displays the model accuracy.
- Generates the confusion matrix and classification report.
